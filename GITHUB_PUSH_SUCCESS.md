## üîß Troubleshooting

### If Deployment Fails

**Common Issues:**

1. **Python Version Error**
   - Solution: Set Python version to 3.11 in Render settings

2. **TensorFlow Installation Error**
   - Solution: Verify requirements.txt has `tensorflow==2.15.0`

3. **Port Binding Error**
   - Solution: Update app.py to use `port=int(os.environ.get('PORT', 5000))`

4. **Model File Not Found**
   - Solution: Verify face_emotionModel.h5 is in the repository

5. **Build Timeout**
   - Solution: Increase build timeout in Render settings

**For detailed troubleshooting, see:** `DEPLOYMENT_CHECKLIST.md`
